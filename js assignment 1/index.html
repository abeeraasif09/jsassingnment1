4. What is the best way to remove a line of code from running that you might want to keep as you debug?

Answer

Locate the line of code you want to disable
Place two forward slashes // at the beginning of the line. For example: Placing // at the start of a line turns it into a comment, and the JavaScript interpreter will ignore it.
Save your code.
By commenting out a line of code, you effectively "disable" it temporarily. This allows you to test your code without that particular line of code running. When you're done debugging and want to re-enable the line, simply remove the // characters, and the code will run as usual.
